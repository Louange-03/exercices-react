# Exercices React 

Ce dépôt contient mes exercices réalisés dans le cadre de l’apprentissage de React.

##  Exercices inclus

###  Exercice 1 — Toggle avec compteur d'effets
- Utilise deux états : `isActive` et `counter`
- `useEffect` dépendant de `isActive`
- Incrémente le compteur à chaque changement d’état
- Fichier : [`exo_Toggle.html`](./exo_Toggle.html)

###  Exercice 2 — Exercice météo
- Utilise `useState` et `useEffect`
- Simule un chargement avec `setTimeout(1s)`
- Affiche un tableau “Heure / Température (°C)”
- Fichier : [`exo_meteo.html`](./exo_meteo.html)

##  Technologies utilisées
- **React**
- **TailwindCSS**

## Pour exécuter les exercices
1. Télécharger ou cloner ce dépôt.
2. Ouvrir les fichiers `.html` dans un navigateur (par double-clic ou via VS Code avec "Go Live").
